InputSimulatorCpp
===============

A Cross Platform (Linux/Mac/Windows) Input Simulator written in C++.

The primary purpose of this library:

**Mouse**

- Move
- Clicks
- Scrolling of the mouse wheel

**Keyboard**

- Press
- Release
- Key Stroking

## Support Platform

| Platform       | Status |
|----------------|--------|
| Windows        | ✅      |
| macOS          | ✅      |
| Linux(X11)     | ✅      |
| Linux(Wayland) | ❌      |
| Other          | ❌      |
